Подробное описание

Класс User_Interface.

Для взаимодействия сервера с командной строкой предназначены методы: operator для запуска сервера и User_Interface для обработки параметров командной строки и UI_err для обработки ошибок парсера

Предупреждения
Все параметры для ввода являются обязательными
